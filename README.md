# pstat100project
# Data Analytics final project on public Airbnb listings data for Oakland California.

This project analyzes Airbnb listing prices in Oakland, California using a cleaned listings dataset from Inside Airbnb. The goal is to understand which listing
characteristics are most associated with price and to evaluate whether common Airbnb features such as room type, neighbourhood, availability, minimum nights,
review activity, and host listing count help explain differences in listing prices. This topic is useful because Airbnb prices vary widely, and understanding the factors
related to price can help identify broader patterns in short-term rental markets.
